# What is this?
This is a npm package that gets random quots!

# installation
`npm i Quote-api`

# How to use :
```
const { randomQuote, randomTen } = require('Quote-api')

console.log(randomQuote)
```
## Options
 quote : Get the quote.
 by : Get who wrote that quote.
 id : Get the number of the quote.
